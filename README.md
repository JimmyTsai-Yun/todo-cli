# todo

A simple command-line todo list manager written in Python.

## Usage

```bash
python todo.py --help
```

## Commands

| Command | Description |
|---|---|
| `todo add "<task>"` | Add a new task |
| `todo list` | List all tasks |
| `todo done <id>` | Mark a task as done |
| `todo delete <id>` | Delete a task |

## Data

Tasks are stored in `~/.todo.json`.
